Introducción

El proyecto Scam LAD es una aplicación web enfocada en la supervisión y administración de cámaras de seguridad. Su propósito es ofrecer una herramienta eficiente para el monitoreo en tiempo real, almacenamiento de grabaciones y gestión de alertas, mejorando la seguridad en entornos residenciales, comerciales o institucionales.

Planteamiento del Problema

En la actualidad, muchas instituciones y hogares carecen de un sistema centralizado para el control y seguimiento de cámaras de seguridad. Esto genera dificultades para detectar incidentes a tiempo, así como una deficiente administración de las grabaciones y la información capturada por los dispositivos.

Justificación

La implementación de Scam LAD permite optimizar la vigilancia y el resguardo de espacios mediante una plataforma accesible y fácil de usar. Este sistema busca integrar tecnología moderna que facilite el control de cámaras, la visualización remota y la respuesta oportuna ante eventos de riesgo.

Objetivos

  • Desarrollar una aplicación para el monitoreo y gestión de cámaras de seguridad.
  • Facilitar la supervisión en tiempo real desde distintos dispositivos.
  • Implementar alertas automáticas ante eventos sospechosos o movimientos detectados.
  • Garantizar la protección y privacidad de los datos almacenados.
